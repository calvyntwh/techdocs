# genesis

genesis：创世工具，生成创世文件_genesis.json

## 1. 使用方法

命令运行格式如下：

```
Usage:
  genesis --chainid=local --path=/data/genesis/local

Flags:
  -h, --help   help for genesis
```

## 2. 命令详解

- **command**

  ```
  genesis --chainid=local --charter=/data/genesis/local
  ```

  | chainid  |  String  | 链id                                                         |
