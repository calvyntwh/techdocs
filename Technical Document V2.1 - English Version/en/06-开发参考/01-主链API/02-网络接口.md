## 1. abci\_info

向BCBChain的节点查询 **bcchain** 世界状态信息。

**. Request URI over HTTP/HTTPS**

```
http://earth.bcbchain.io/abci_info
or
https://earth.bcbchain.io/abci_info
```

**.  Request JSONRPC over HTTP/HTTPS**

```
{
  "jsonrpc": "2.0",
  "id": "dontcare/anything",
  "method": "abci_info"
}
```
